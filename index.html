Photography Studio Demos

We have prepared five highly distinct visual and functional concepts for the photo studio client. Each is isolated in its own folder, utilizing professional React frontend layouts and unique CSS design systems.

01 /

Aether Fine Art

Minimalist and editorial layout designed for gallery exhibitions, fine-art prints, and portraiture. Large whitespace, quiet transitions, B&W filters.

Palettes Linen & Charcoal
Typography Cormorant Garamond (Serif), Inter (Sans)
Core Tech Ambient Theme Toggle, Lightbox Zoom, EXIF Display
cd aether; npm run dev
02 /

Everglow Wedding

Warm, welcoming, and storytelling focus tailored for weddings, lifestyle couples, and organic family portraits. Narrative timelines, warm tones.

Palettes Ivory, Rose & Sage
Typography Lora (Warm Serif), Montserrat (Friendly Sans)
Core Tech Coverage Calculator Slider, Story Timeline, Reviews
cd everglow; npm run dev

cd prism; npm run dev
